Add website images here.

Folders:

- `home`
  - homepage hero images, trust sections, warehouse photos, packaging shots
- `services`
  - service page visuals for sourcing, warehousing, packing, shipping
- `proof`
  - QC screenshots, tracking examples, stock screenshots, operational proof
- `blog`
  - blog banners and article cover images
- `catalog`
  - product category and catalog card images
- `logos`
  - partner logos, badges, supported platform marks
- `testimonials`
  - customer avatars, store logos, case study visuals

Recommended file naming:

- use lowercase
- use hyphens
- keep names descriptive

Examples:

- `home/warehouse-shenzhen-01.jpg`
- `proof/qc-check-example.png`
- `services/shopify-fulfillment.jpg`
- `blog/china-sourcing-guide-cover.jpg`

Public URL pattern:

- `/site-assets/home/warehouse-shenzhen-01.jpg`
- `/site-assets/proof/qc-check-example.png`

Preferred formats:

- photos: `.jpg` or `.webp`
- UI screenshots: `.png`
- logos with transparency: `.png` or `.webp`
